# Bamboozled-quiz-app
A fun and interactive quiz web app built with HTML, CSS, and JavaScript. The app fetches questions dynamically from QuizAPI, allowing a fresh and varied quiz experience each time. As you play, the app instantly shows whether your answer is correct or not and keeps track of your score.
